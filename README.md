
# Interprocess Communication b/w Golang & Java

This a collection of code samples using the various protocols by which the communicaiton can be established.

Samples are present for the following protocols
    
    Simple Socket
    SSL secured socket
    Unix Sockets
    gRPC
    
## Tech Stack

**Client:** Java, Golang

**Server:** Java, Golang
